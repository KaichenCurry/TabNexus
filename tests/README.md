# Tests and evaluation data

- Root `*.test.*` files cover application logic, UI, storage, Chrome APIs, security contracts, and MCP behavior.
- `e2e/` contains the loaded-extension Chrome scenarios.
- `fixtures/m0/` contains synthetic M0 clustering samples.
- `evals/tabnexus-mcp/` contains the curated MCP dataset and deterministic capability reports.

Use `pnpm check` for the complete local validation pipeline.
